< 실습문제 1. > 아래 IP 주소를 보고 클래스, 네트워크 주소, 호스트 주소를 구분하여 작성하시오.

- 1번: 10.24.5.1 =>  A클래스 / 10 / 24.5.1
- 2번: 192.233.1.2 =>  C클래스 / 192.233.1 / 2
- 3번: 132.14.200.5 =>  B클래스 / 132.14 / 200.5
- 4번: 203.122.50.22 =>  C클래스 / 203.122.50 / 22
- 5번: 180.255.255.255 =>  B클래스 / 180.255 / 255.255



< 실습문제 2. > 아래 IP 주소를 보고 클래스, 서브넷 마스크, 서브넷 네트워크, 호스트 주소의 개수를 구분하여 작성하시오.

*서브넷 마스크가 직접적으로 주어지지 않는다면 A클래스 8 / B클래스 16 / C클래스 24이다.
*서브넷 네트워크 : IP 주소랑 서브넷 마스크를 AND 연산해서 나온 결과값.
*호스트 주소의 개수는 서브넷 마스크 0의 개수로 구하기

- 1번: 10.24.5.1/18 
 A클래스 / 255.255.192.0 / 10.24.0.0 / 2^14-2

- 2번: 192.233.1.2/27
C클래스 / 255.255.255.224 / 192.233.1.0 / 2^5-2

- 3번: 132.14.200.5
B클래스 / 255.255.0.0 / 132.14.0.0 / 2^16-2

- 4번: 203.122.50.22/30
C클래스 / 255.255.255.252 / 203.122.50.20 / 2^2-2

- 5번: 180.255.255.255
B클래스 / 255.255.0.0 / 180.255.0.0 / 2^16-2
